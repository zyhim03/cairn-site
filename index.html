Where Cairn fits

Between too little and too much.

The default

Spotlight

Built into macOS. Apps, filenames, and the basics. Reads inside common file types through its background importers, but content search isn’t where it puts its weight.

  • Apps and filenames
  • Web and dictionary suggestions
  • Quick calculations
  • Built into macOS
  • Privacy: Apple, stays on device
  • Local mode: always
For beginners
The middle

Cairn

Searches the words written inside your files. Fuzzy matched, ranked by relevance, with the line each match lives on. Fast, local, focused. Stops where complexity creeps in.

  • Searches inside file contents, ranked, with the matching line shown
  • Forgiving fuzzy matching
  • Math and currency answers
  • Folder organization suggestions
  • Privacy: never leaves your Mac
  • Local mode: no account, no cloud
For everyone in between
The toolkit

Raycast

A launcher with hundreds of extensions, an AI assistant, scripting, and window management. Powerful, but a lot to learn.

  • Extensions marketplace
  • AI assistant (Pro)
  • Script commands and snippets
  • Window management
  • Privacy: cloud sync required
  • Local mode: limited, account required
For power users
